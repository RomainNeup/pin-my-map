# Pin my map

This is a simple app to save your favorite places on a map.

## Environment variables

### Frontend

- `PUBLIC_MAPBOX_ACCESS_TOKEN`: Mapbox token
- `PUBLIC_API_BASE_URL`: API base URL

### Backend


## How to run

### Frontend

```bash
cd front
yarn install
yarn start
```

### Backend

```bash
cd back
yarn install
yarn start
```
